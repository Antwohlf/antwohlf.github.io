Cosas Interesantes

Esta página es una lista en constante expansión de cosas con las que me he topado en Internet. Estoy seguro de que encontrarás algo interesante, pero si no, hazme saber qué tipo de cosas interesantes puedo añadir aquí.

La tecnología es genial

  • La Biblioteca de Babel - Esta biblioteca contiene todas las combinaciones posibles de letras y palabras. La siguiente canción de tu artista favorito, la declaración de independencia, una descripción detallada del mejor día de tu vida - todo está en alguna parte. Además, páginas interminables de letras aleatorias
  • Esta Persona No Existe - Este sitio web utiliza una red neuronal para generar una cara de alguien que no existe
  • Esta Inicio No Existe - Al igual que el sitio anterior, utiliza una red neuronal para generar un sitio web de inicio falso.
  • Escucha Wikipedia - Este sitio web te muestra para escuchar y ver las últimas ediciones en Wikipedia

Useful tools

  • ¿Abajo Para Todos o Sólo Para Mí? - Comprueba si un sitio web está realmente caído o si tienes problemas de conexión
  • La Máquina de Wayback - Ver una versión anterior de un sitio web
  • Traductor de DeepL - Una alternativa a Google Translate que utiliza el aprendizaje profundo y la inteligencia artificial para su traductor
  • Let Me Google That For You - Generar un enlace para esa persona que te pregunta algo que fue fácil para Google
  • Rome2Rio - Encuentre una manera de llegar a dos lugares cualquiera, muy útil para viajar

Learn Something New

  • Cómo Funcionan los Vehículos Autónomos - Pequeño sitio informativo que encontré sobre vehículos autónomos
  • Wolfram Alpha - Esto es para las matemáticas, hace todas las matemáticas
  • Cursos Abiertos del MIT - Son clases del MIT, pero gratis
