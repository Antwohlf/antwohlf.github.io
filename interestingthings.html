Interesting Things

This page is an ever-expanding list of things I've stumbled across on the internet. I'm sure you'll find something interesting but if not, let me know what kind of cool things I can add here.

Technology is cool

  • The Library of Babel - This library contains every possible combination of letters and words. Your favorite artist's next song, the declaration of independence, a detailed description of the best day of your life - it's all somewhere in there. Also, endless pages of random letters
  • This Person Does Not Exist - This website uses a neural network to generate a face of someone that doesn't exist
  • This Startup Does Not Exist - Similar to the previous site, this uses a neural network to generate a fake startup website
  • Listen to Wikipedia - This website shows you to listen to/watch the latest edits on Wikipedia

Useful tools

  • Down For Everyone or Just Me? - Check if a website is actually down or if you're having connection problems
  • The Wayback Machine - Look at an older version of a website
  • DeepL Translator - An alternative to Google Translate that uses deep learning and artificial intelligence for its translator
  • Let Me Google That For You - Generate a link for that person who asks you something that was easy to Google
  • Rome2Rio - Find a way to get between any two places, very useful for traveling

Learn Something New

  • How Self Driving Cars Work - Informative little site that I found about autonomous vehicles
  • Wolfram Alpha - This is for math, it does all of the math
  • MIT Open Courseware - It's MIT classes, but for free

Entertainment

  • For the Sparty fans
  • Halo Homepage - If you've ever played Halo, you'll enjoy this page
  • XKCD - A webcomic of romance, sarcasm, math, and language
  • Hackertyper - This is where you pretend to be a hacker like in the movies
  • GeoGuessr - This site drops you in a random location on Google Maps, and you have to look around and try to guess where you are
